This Website uses React, JavaScript & HTML.
This was built as a demonstration of my knowledge so far.

I experimented with Prime React (https://primereact.org/) for its components and ease of styling.
Key features:
- Photo Gallery is lazy loaded for optimization when not in view
- We check the time of day to show if we are opened or closed
- The page is suited for Mobile UI and Web Browser UI
- The quote system submits to a MySQL Database and allows File Uploads
- Photo Gallery has a "tag" system for easy sorting.
- Photo Gallery has a "Scroll to Top" button to help when the page gets long
